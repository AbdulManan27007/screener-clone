# Screener-Clone

# Prerequisites
Before you begin, make sure you have the following installed:

1. Node.js
2. npm (or pnpm if you're using it)

# Installation
1.  Navigate to the client folder:
cd client
npm install


2.  Navigate to the server folder:
cd server
npm install

# Running the Project
1. Navigate to the client folder:
cd client
npm start

2.  Navigate to the server folder:
cd server
npm start
